== Description ==
This plugin allow you to accept payments using Paytm in WPeCommerce. This plugin will add a Paytm Payment option on WPeCommerce checkout page, when user choses Paytm as Payment Method, he will redirected to Paytm website to complete his transaction and on completion his payment, paytm will send that user back to your website along with transactions details. This plugin uses server-to-server verification to add additional security layer for validating transactions. Admin can also see payment status for orders by navigating to Dashboard > Store Sales from menu in admin.

== Installation ==
* Unzip the "WPeCommerce Paytm Payment" module files
* Copy the file "paytm.php" and "paytm" folder to your WordPress installation in this folder: /wp-content/plugins/wp-e-commerce/wpsc-merchants/
* Log in to your WordPress administration
* Go to Settings -> Store
* Choose Payments at the top of the screen, and tick off Paytm. Press "Update" to save the settings.
* After module Installation, configure it.
* You should update Paytm configuration values provided by Paytm on configuration tab.
* Then click on Update.
* Your Paytm WPeCommerce plugin is now setup. You can now accept payment through Paytm.
